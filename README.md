## Tabel Widget & Fungsinya

| Nama Elemen / Lokasi                  | Widget                | Fungsi dalam Aplikasi |
|---------------------------------------|------------------------|------------------------|
| **MaterialApp** (main.dart)           | `MaterialApp`          | Widget utama yang mengatur tema, style, dan halaman awal aplikasi. |
| **Scaffold** (Semua screen)           | `Scaffold`             | Kerangka dasar halaman (AppBar, Body, dan struktur Material Design). |
| **AppBar**                            | `AppBar`               | Header halaman yang menampilkan judul dan tombol back otomatis. |
| **SafeArea**                          | `SafeArea`             | Menjaga konten agar tidak tertutup notch, kamera, atau navigation bar. |
| **Column & Row**                      | `Column`, `Row`        | Column menyusun widget secara vertikal, Row secara horizontal. |
| **Card** (Kategori & Produk)          | `Card`                 | Kotak UI dengan shadow & rounded corner untuk kategori dan produk. |
| **GridView.builder** (ProductList)    | `GridView.builder`     | Menampilkan list produk dalam bentuk grid 2 kolom yang bisa discroll. |
| **InkWell**                           | `InkWell`              | Membuat widget bisa diklik dengan efek ripple (kategori & produk). |
| **Container**                         | `Container`            | Pembungkus styling: padding, background, dekorasi, border radius. |
| **Icon**                              | `Icon`                 | Menampilkan ikon Material Design untuk kategori dan produk. |
| **Text**                              | `Text`                 | Menampilkan teks: judul, nama, label, dan harga produk. |
| **ElevatedButton** (Detail Screen)    | `ElevatedButton.icon`  | Tombol "Kembali" dengan icon panah. |
| **Navigator.push & pop**              | `Navigator`            | `push()` membuka halaman baru; `pop()` kembali ke halaman sebelumnya. |
| **Padding & SizedBox**                | `Padding`, `SizedBox`  | Padding memberi jarak dalam; SizedBox memberi jarak antar widget. |
